# 42-Project---A_Maze_ing
Repo for group project A_Maze_ing

Usage: python3 a_maze_ing.py config.txt

Output will be in maze.txt

Maze is in grid format: each wall represented by NESW, open is 0, closed is 1

an all-enclosed cell is represented by 1111

a maze will be build using recursive DFS algo and printed in hex as required
